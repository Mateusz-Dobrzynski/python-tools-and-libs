# Interactive Jupyter notebooks

## Setup

1. Create a new new notebook (`Ctrl+Shift+P` > Create: New Jupyter Notebook).
2. Select a kernel (.venv).
3. Install dependencies if needed.

## Basic usage
- Press `Ctrl+Enter` to run a cell.
- Create Markdown or code cells.
